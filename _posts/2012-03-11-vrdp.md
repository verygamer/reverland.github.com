## 安装vrdp功能 ##

先确定vbox版本

{% highlight bash %}
vboxmanage -v
{% endhighlight %}

先到官网下载对应的extpack，然后运行(以4.0.8为例)
{% highlight bash %}
sudo vboxmanage extpack install Oracle_VM_VirtualBox_Extension_Pack-4.0.8-71778.vbox-extpack
{% endhighlight %}
## 使用vrdp功能 ##
后台运行虚拟机系统（如xp）
{% highlight bash %}
vboxheadless -s xp
{% endhighlight %}

最后返回一行
{% highlight bash %}
VRDE server is listening on port 3389.
{% endhighlight %}
打开rdp软件比如说krdc连接宿主机ip+3389(默认)

如果在本机桥接vbox仿问虚拟机可以桥接，需要加载相应模块。

{% highlight bash %}
modprobe vboxnetflt
{% endhighlight %}
